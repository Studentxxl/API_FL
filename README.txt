Установка:
копировать файлы проекта на локальный комьпьютер
(чтобы не менять пути, копировать в C:\API_FL)

WIN+R
cmd

cd папка проекта

Установка виртуального окружения (вместо py может быть python3)
py -m venv .venv

Актвация виртуального окружения
.venv\Scripts\activate.bat

Установка зависимостей
pip install -r requirements.txt

Если проект установлен не в C:\API_FL
заменить пути на свои в файлах server_start.bat и test_start.bat

Параметры тестов настраиваются в test.py в этом блоке:
# ****************************
# БЛОК ВЫЗОВА ТЕСТОВЫХ ФУНКЦИЙ
# ****************************

Запуск сервера
server_start.bat

Запуск теста
test_start.bat
